##Task Management Application

# Overview
A command-line task management system built in Python that helps users organize, track, and manage their daily tasks efficiently. The application provides comprehensive task tracking with priority management, status monitoring, and productivity analytics.

# Core Functionality
- **Add Tasks**: Create new tasks with custom titles and priority levels (High/Medium/Low)
- **View Tasks**: Display all tasks with their ID, title, status, and priority
- **Complete Tasks**: Mark tasks as completed with a single command
- **Delete Tasks**: Remove unwanted tasks from the system
- **Edit Tasks**: Modify existing task titles and priorities
- **Filter Tasks**: View tasks by status (pending/completed)
- **Productivity Statistics**: Generate comprehensive reports showing completion rates

## Task Management Features
- Unique ID assignment for each task
- Three-level priority system (High=1, Medium=2, Low=3)
- Status tracking (pending/completed)
- Task editing capabilities
- Productivity metrics and reporting

## Technologies/Tools Used
- **Language**: Python 3.x
- **Data Structure**: Lists and Dictionaries for in-memory storage
- **Control Flow**: Menu-driven interface with while loop
- **Input Handling**: Built-in input() and validation

### Prerequisites
- Python 3.6 or higher installed on your system
